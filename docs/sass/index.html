    mixins

    mq-min

    Since 1.0.0
    @mixin mq-min($key) { ... }

    Description

    ブラウザ幅またはviewportの幅が、指定したブレークポイントの最小値以上の場合にのみ、スタイルを適用する
    指定したブレークポイントの最小値が0の場合は、すべてにスタイルを適用する

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $key

    ブレークポイント

    		String—none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    .element {
	@include mq-min("sm") {
		width: 100%;
	}
}

// CSS Output
@media (min-width: 480px) {
	.element {
		width: 100%;
	}
}

    Requires

    • [function] breakpoint-min
    • [variable] breakpoints

    sprite

    Since 1.0.0
    @mixin sprite($key, $scale: 1) { ... }

    Description

    スプライト用のCSSを出力する

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $key

    スプライト名（フォルダ名-画像ファイル名）

    Example

	background-position: 0px 0px;
	background-size: 262px 262px;
	width: 128px;
	height: 128px;
