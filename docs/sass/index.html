    01. グリッド (flexbox)

    mixins

    grid

    Since 1.1.0
    @mixin grid($options) { ... }

    Description

    グリッドの設定を変更する
    設定の変更は、ネストしたスコープ内でのみ有効

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $options

    $base-grid-optionsを参照

    		Map—none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    .row {
	@include grid(("gutters": false)) {
		@include row {
			.col {
				@include col;
			}
		}
	}
}

// CSS Output
.row {
	display: flex;
}
.row .col {
	flex: 0 0 auto;
	max-width: 100%;
	flex-grow: 1;
	flex-basis: 0;
}

    Requires

    • [mixin] grid-set
    • [mixin] grid-reset

    row

    Since 1.1.0
    @mixin row($options...) { ... }

    Description

    コンテナーとして適用する

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $options...

    コンテナーのオプション
    値からFlexboxコンテナーのプロパティを適用

    値プロパティ名プロパティ値
    rowflex-directionrow
    reverserow-reverse
    startjustify-contentflex-start
    centercenter
    endflex-end
    betweenspace-between
    aroundspace-around
    topalign-itemsflex-start
    middlecenter
    bottomflex-end
    baselinebaseline
    stretchstretch

    		可変長引数—none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    .row {
	@include row(center, middle) {
		.col {
			@include col;
		}
	}
}

// CSS Output
.row {
	-ms-flex-pack: center;
	    justify-content: center;
	-ms-flex-align: center;
	    align-items: center;
	display: -ms-flexbox;
	display: flex;
	box-sizing: border-box;
	margin-right: -1%;
	margin-left: -1%;
}
.row .col {
	-ms-flex: 0 0 auto;
	    flex: 0 0 auto;
	max-width: 100%;
	-ms-flex-positive: 1;
	    flex-grow: 1;
	-ms-flex-preferred-size: 0;
	    flex-basis: 0;
	box-sizing: border-box;
	padding-right: 1%;
	padding-left: 1%;
}

    col

    Since 1.1.0
    @mixin col($columns: null, $offset: null, $order: null) { ... }

    Description

    カラムとして適用する

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $columns

    カラムの幅として設定するグリッドカラム数
    nullの場合は、コンテナーに応じて各カラムが均等に調整される
    autoの場合は、カラムのコンテンツに応じて調整される

    		String or Number or Nullnull
    $offset

    カラムのオフセットとして設定するグリッドカラム数

    		Number or Nullnull
    $order

    カラムの順番

    		Number or Nullnull

    Example

    .col-1 {
	@include col(8, 4, 2);
}
.col-2 {
	@include col(12, null, 1);
}

// CSS Output
.col-1 {
	-ms-flex: 0 0 auto;
	    flex: 0 0 auto;
	max-width: 33.33333%;
	-ms-flex-preferred-size: 33.33333%;
	    flex-basis: 33.33333%;
	margin-left: 16.66667%;
	-ms-flex-order: 2;
	    order: 2;
	box-sizing: border-box;
	padding-right: 1%;
	padding-left: 1%;
}
.col-2 {
	-ms-flex: 0 0 auto;
	    flex: 0 0 auto;
	max-width: 50%;
	-ms-flex-preferred-size: 50%;
	    flex-basis: 50%;
	-ms-flex-order: 1;
	    order: 1;
	box-sizing: border-box;
	padding-right: 1%;
	padding-left: 1%;
}

    has

    Since 1.1.0
    @function has($list, $values, $operation: "AND") { ... }

    Description

    リストに指定した値が含まれているか確認する

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $list

    検索対象のリスト

    		List—none
    $values

    検索条件

    		String or Number or Color or List—none
    $operation

    $valuesにリストを渡した場合の動作

    値説明
    "AND"$valuesの全てが$listに含まれる場合にtrueを返す
    "OR"$valuesのいずれかが$listに含まれる場合にtrueを返す

    		String"AND"

    Returns

    Boolean

    Example

    $list: ("A" "B" "C");
@debug has($list, "A"); // true
@debug has($list, ("A" "B")); // true
@debug has($list, ("A" "Z")); // false
@debug has($list, ("A" "Z"), "OR"); // true
@debug has($list, ("Y" "Z"), "OR"); // false

    map-get-prev-key

    Since 1.0.0
    @function map-get-prev-key($map, $key) { ... }

    Description

    マップから、指定したキーの前の要素のキーを取得する
    指定したキーが、マップの最初の要素である場合は、nullを返す

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $map

    操作対象のマップ

    		Map—none
    $key

    取得対象要素の次の要素のキー

    		String—none

    Returns

    String or Null —

    取得対象要素のキー、またはnull

    Example

    $map: ("A": 1, "B": 2, "C": 3);
@debug map-get-prev-key("B", $map); // "A"
@debug map-get-prev-key("A", $map); // null
