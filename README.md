# Website Coding Kit

### Dependencies

- Node.js (>=4.4)
- Gulp (>=3.9) `Option`
- Bower (>=1.7) `Option`

### Setup

```
npm run init
```

### Usage

#### Build

**Run before you start development.**

```
npm run build
```

Production Build

```
npm run build -- --env=production
```

#### Watch file / Start server

```
npm run start
```
